# Configuration

This page goes over the settings.json configuration.

- [`Ubuntu`](#ubuntu-and-debian) → `>=20.04`
- [`Debian`](#ubuntu-and-debian) → `>=10`
- [`Windows`](#windows) → `>=10`

```json
{
  "api": {
    "token": "Termuibabjfgbnggggg",
    "_comment": "Dont change *stObj, Unless You dont know about it!"
  },
  "app": {
    "PORT": 80,
    "appLink": "http://cdn.nicoruiz.dev",
    "appName": "Content Delivery Network",
    "appFavicon": "https://avatars.githubusercontent.com/u/77137417?v=4"
  },
  "display_embed": {
    "title": [
      "NicoRuizDev/DiscordCDN",
      "Content Delivery Network | NicoRuizDev"
    ],
    "description": ["Nothing is free", "Nothing is free"]
  }
}
```

#### PORT:

Replace it with the allowed port. (eg ip:PORT)

#### appLink:

Replace it with your server ip or domain. (eg http://cdn.nicoruiz.dev)

#### appName:

Replace it with your application name.

#### appFavicon:

Replace it with your application logo link.

#### title:

Replace it with your own appearance and creativity. It will be shown in the title of the discord embed.

#### description:

Description will also be shown in the discord embed change if you want to.
