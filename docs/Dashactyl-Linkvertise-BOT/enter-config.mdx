# Configuration

This page goes over the .envexample configuration:

Firstly, rename the ".envexample" to ".env"

```bash
mv .envexample .env
```

Then, open the .env file:

```bash
nano .env
```

You will see some of the configurations like:

```envexample
DISCORD_TOKEN=
APP_NAME=
APP_ICON=
APP_URL=
DASH_URL=
DASH_API=
LINKVERTISE_USER_ID=
COINS_TO_GIVE=
COOLDOWN=
```

#### DISCORD_TOKEN:

Replace it with your bot token

#### APP_NAME:

Replace it with your app name (eg: switchmanager)

#### APP_ICON:

Replace it with your app icon (eg: APP_ICON: https://cdn.discordapp.com/icons/905747656045912064/36a39621db3580ac133006d7150e5952.png)

#### APP_URL:

Replace it with your app url (eg: earn.nicoruiz.dev)

#### DASH_URL:

Replace it with your dashbord url (eg: dash.nicoruiz.dev)

#### DASH_API:

Replace it with your dash api

#### LINKVERTISE_USER_ID:

Replace it with your linkvertise user id

#### COINS_TO_GIVE:

Replace it with how much coins to give

#### COOLDOWN:

Replace it with how much cooldown you want
