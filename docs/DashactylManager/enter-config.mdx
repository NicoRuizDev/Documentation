# Configuration

This page goes over the .envexample configuration:

Firstly, rename the ".envexample" to ".env"

```bash
mv .envexample .env
```

Then, open the .env file:

```bash
nano .env
```

You will see some of the configurations like:

```json
Name=
Icon=
Dash_URL=
Admin_ROLE_ID=
DASH_API=
Token=
DISCORD_CLIENT_ID=
```

#### Name:

Replace it with your bot name (eg: switchmanager)

#### Icon :

Replace it with your bot icon (eg: icon.png)

#### Dash_URL:

Replace it with your dashbord url (eg: dash.example.com.)

#### Admin_ROLE_ID:

Replace it with your admin role id (eg: 994498195575025695)

#### DASH_API:

Replace it with your dashactyl api

#### Token:

Replace it with your bot token

#### DISCORD_CLIENT_ID:

Replace it with you discord client id
